## 任務內容
+ 建立一個可放入一個參數的函數，此參數需有預設值[[5,3,1,2,4],[3,4,5,1,2]]
+ 此函數會將放入的參數進行排列
  1. 奇數欄位將排列成[1,2,3,4,5]
  2. 偶數欄位將排列成[5,4,3,2,1]
+ 最後回傳結果

## Hint:

`list.sort()`用法展示

```python
a = [5,2,3,1,4]

```

```python
a = [5,2,3,1,4]

a.sort(reverse=True)

print(a)
```
